## 🎯 우리가 해결하는 문제

한국어를 공부해도 실제 한국에서 대화하면 막막한 이유:

1. **교재 한국어 ≠ 실제 한국어** — 교과서엔 없는 구어체, 줄임말, 상황별 뉘앙스
2. **AI 챗봇이 너무 어려운 말을 씀** — 초급 학습자한테 고급 단어를 쏟아냄
3. **연습 후 뭘 틀렸는지 모름** — "틀렸어요" 말고 왜 틀렸는지 안 알려줌

## 🗺 사용자 여정 (User Journey)

```
첫 접속
  → 웰컴 화면 (앱 소개 + "맞춤 학습 설정 시작하기" CTA)
  → 맞춤 학습 설정 (/setup) — 국적/닉네임/수준/관심문화/가보고싶은 곳 (최초 1회, UUID 자동 생성)
      → 즉시시작 선택 시: 시나리오 생성 → 역할선택 → 대화
      → 홈으로 선택 시: 홈 화면으로 이동
  → 홈 화면 (TierCard / 주간통계 / 재도전카드 / 복습배너 / CTA "대화 시작하기")
  → 장소 선택 → 시나리오 + 대화상대 자동 생성
  → 역할(페르소나) 선택 — 내가 맡을 역할/미션 선택, 안 고른 쪽이 AI 상대방
  → 대화 시작 (MVP: 초급 7턴, 최소 4턴 후 조기 분석 요청 가능, 한국어 입력 필수)
  → 대화 종료 → "결과 확인하기" 버튼 클릭 → 점수 + 등급(Grade) 확인
  → 피드백 (틀린 부분 빨간 표시 + 오답 단어 목록 + 요약 + 레이더 그래프)
  → 주간 복습 (초성 퀴즈 + 플래시 카드) — 🚧 미구현
```

> **MVP 핵심 제약**: **초급 사용자 × 한강 맵만** 우선 구현

| 주차 | FE 작업 | 우선순위 |
|------|---------|----------|
| 1주차 | 프로젝트 세팅 + 디자인 토큰 + 홈 화면 기본 레이아웃 | 🔴 Must |
| 2주차 | 맞춤 학습 설정 UI + 튜토리얼 | 🔴 Must |
| 3주차 | 채팅 UI + AI 연결 (초급/한강) | 🔴 Must |
| 4주차 | 결과 화면 + 피드백 UI + 방사형 그래프 | 🟡 Should |

| 6주차 | 플래시 카드 + Scaffolding 팝업 + 애니메이션 polish | 🟢 Could |

## 📊 핵심 KPI

> KPI = 서비스가 잘 되고 있는지 측정하는 수치 목표

### KPI 1. 대화 완주율
- **정의**: 대화 시작 후 끝까지 완료한 비율
- **목표**: MVP 기준 **60% 이상**
- **측정**: 대화 완료 수 ÷ 대화 시작 수 × 100

### KPI 2. 맞춤 학습 설정 완료율
- **정의**: 첫 접속 후 맞춤 학습 설정을 끝까지 마친 비율
- **목표**: **80% 이상**
- **측정**: 설정 완료 수 ÷ 첫 접속 수 × 100

### KPI 3. 주간 재방문율
- **정의**: 1회 이용 후 다음 주에도 돌아오는 비율
- **목표**: **40% 이상**
- **측정**: 2주차 이상 접속 수 ÷ 전체 가입 수 × 100

### KPI 4. 평균 점수 향상도
- **정의**: 1회차 대비 3회차 평균 점수 차이
- **목표**: **+1.5점 이상** (10점 만점)
- **측정**: 3회차 평균 - 1회차 평균

### KPI 5. 피드백 확인율
- **정의**: 대화 완료 후 피드백 화면 진입 비율
- **목표**: **70% 이상**
- **측정**: 피드백 진입 수 ÷ 대화 완료 수 × 100

## 🏗 기술 구조 요약 (FE 관점)

```
사용자 브라우저 (Next.js 15 + React 19 — 이성준 담당)
        ↕ REST API (v1/)
BE 서버 FastAPI + LangGraph (이강산 담당)
        ↕
LLM (OpenAI) + 벡터DB (ChromaDB) + DB (PostgreSQL)
```

**FE 핵심 라이브러리**: Tailwind CSS v4, Recharts (레이더 그래프), Lucide React (아이콘)
**상태 관리**: localStorage (유저 프로필, UUID), sessionStorage (채팅 세션)
**필드 네이밍**: BE snake_case 기준으로 FE 전체 통일 완료 (2026-04-02)
**BE↔FE 매핑 상세**: `docs/API_MAPPING.md` 참조

## ⚠️ 기획서 원본과의 주요 차이점

| 항목 | 기획서 원본 | 현재 확정 |
|------|------------|-----------|
| MVP 장소 | 8개 전체 | **한강만** |
| MVP 난이도 | 초/중/고 전체 | **초급만** |
| 장소 목록 | 고정 8개 | **미확정, 추후 추가 가능** |
| 미디어 경험 질문 | 예/아니오 | **6가지 문화 카테고리** 버튼 선택 (`cultural_interest`) |
| 국적 입력 | 텍스트 or 드롭다운 | 드롭다운 + IP 자동선택 (FE 처리) |
| 홈 화면 | 없음 | 신설 (등급카드/통계/재도전/복습배너/탭바) |
| 튜토리얼 | 없음 | 기존 스포트라이트 제거, **재설계 대기** |
| 레이아웃 | 미정 | 모바일 고정 max-width 480px |
| 유저 식별 | 닉네임+코드 | **UUID** (`crypto.randomUUID()`) |
| 등급 체계 | `tier` | **`grade`** (Bronze/Silver/Gold/Platinum/Diamond) |
