## 주요 의사결정

### 1. 페르소나 = 사용자의 역할
- **기존**: 페르소나를 AI 대화 상대로 취급 ("이 사람과 대화 시작")
- **변경**: 사용자가 선택한 페르소나가 되어 AI 상대방과 대화 ("이 페르소나로 대화 시작")
- **영향**: persona/page에서 선택한 것 = 내 역할, 안 고른 것 = 상대방으로 sessionStorage 전달

### 2. 라이트 배경 화이트 전환
- **기존**: #F5F0E8 베이지 → 따뜻하지만 card-bg(#FFFFFF)와 구분이 약함
- **변경**: background #FFFFFF, card-bg #F7F7F7 → 깔끔한 화이트 베이스에 연회색 카드

### 3. 다크 골드 색상 조정
- 최초 #FAC775 → 채도 -20% → 밝기 -20% → +10% → -3% → -3% = 최종 **#D0A95C**
- setup 계열도 동일 비율: accent #B7933B, hover #B0904A

### 4. 결과 이동 방식 변경
- **기존**: 대화 종료 1.5초 후 자동 이동
- **변경**: "결과 확인하기" 버튼 표시 → 사용자가 직접 클릭
