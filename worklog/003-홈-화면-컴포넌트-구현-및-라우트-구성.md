## 기술 용어 주석

- **`usePathname()`**: Next.js에서 현재 페이지의 URL 경로를 읽어오는 훅. `"use client"` 선언이 있는 컴포넌트에서만 사용 가능.
- **`"use client"`**: Next.js의 서버 컴포넌트 vs 클라이언트 컴포넌트 구분 지시어. `useState`, `useEffect` 같은 브라우저 전용 기능을 쓸 때 파일 맨 위에 선언.
- **`fixed bottom-0`**: CSS position: fixed. 스크롤해도 항상 화면 같은 위치에 고정.
- **`z-50`**: z-index 값. 다른 요소 위에 겹쳐 표시되도록 쌓임 순서를 높임.
- **`active:scale-95`**: Tailwind 클래스. 버튼을 누르고 있는 동안(active 상태) 95% 크기로 줄어드는 효과.
- **유니온 타입 (`"Bronze" | "Silver" | ...`)**: TypeScript에서 정해진 값 중 하나만 허용하는 타입 선언 방식. `tier: "Bronze"` 처럼 오타가 있으면 빌드 시 에러 발생.
- **`Record<K, V>`**: TypeScript 유틸리티 타입. 키 타입 K, 값 타입 V인 객체를 정의. `Record<Tier, string>`은 Tier의 모든 키에 string 값이 있어야 함.
- **Placeholder 페이지**: 아직 본격 구현 전이지만, 라우트가 존재해야 탭 링크가 에러 없이 동작하므로 미리 빈 페이지를 만들어 두는 것.
