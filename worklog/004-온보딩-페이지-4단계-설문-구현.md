## 기술 용어 주석

- **TDD (Test-Driven Development)**: 테스트를 먼저 작성하고, 그 테스트를 통과하도록 코드를 구현하는 개발 방식. 나중에 코드를 바꿔도 테스트가 망가지는 걸 바로 발견할 수 있음.
- **Jest**: JavaScript/TypeScript 테스트 프레임워크. `test("설명", () => { 검증 })` 형태로 작성.
- **ts-jest**: TypeScript 파일을 Jest가 이해할 수 있도록 변환해주는 도구.
- **`as const`**: TypeScript 키워드. 배열/객체를 수정 불가능한 리터럴 타입으로 고정. 타입 추론을 더 정확하게 해줌.
- **`router.replace()`**: 페이지를 이동하되, 브라우저 히스토리에 이전 페이지를 남기지 않음. 뒤로 가기 버튼을 눌러도 온보딩으로 돌아오지 않게 할 때 사용.
- **`ipapi.co`**: IP 주소를 기반으로 사용자의 국가를 감지해주는 무료 외부 API. FE에서 직접 호출하므로 BE 불필요.
- **드롭다운 외부 클릭 감지**: `useRef`로 드롭다운 DOM 요소를 참조 → `mousedown` 이벤트에서 클릭 위치가 드롭다운 바깥인지 확인 → 바깥이면 닫기.
- **커스텀 훅 (`use` 접두사)**: React 훅(useState 등)을 조합한 재사용 가능한 로직 묶음. `use`로 시작하는 함수여야 React가 훅 규칙을 적용함.
