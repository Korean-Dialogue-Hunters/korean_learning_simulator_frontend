# 011 — 닉네임 입력 + 온보딩 5단계 확장

## 작업 범위
- 온보딩(맞춤 학습 설정) 국적 선택 다음에 닉네임 입력 단계 추가
- 4단계 → 5단계로 확장 (국적 → 닉네임 → 수준 → 문화 → 장소)

## 주요 기능

- **닉네임 추천**: 한국어 "귀여운 수식어 + 명사" 조합 (수식어 55개, 명사 75개)
- **랜덤 생성 버튼**: 주사위 아이콘, 누르면 입력란에 바로 적용
- **6자리 코드**: 닉네임 입력 시 우측에 #000000 형태로 항상 표시
- **32바이트 제한**: 실시간 바이트 카운터, 초과 시 빨간 경고
- **테스트**: 15개 (바이트 계산, 닉네임 생성, 코드 생성, 유효성 검사)

---

## 기술 용어 참조

- **바이트(byte)**: 데이터 크기 단위. 영어는 1바이트, 한글은 3바이트(UTF-8 기준)
- **TextEncoder/Buffer**: 문자열을 바이트로 변환하는 도구. 브라우저는 TextEncoder, Node.js(테스트)는 Buffer 사용
- **placeholder**: 입력란이 비어있을 때 보이는 안내 텍스트 (옅은 글씨)
