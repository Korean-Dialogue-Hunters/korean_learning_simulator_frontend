## 기술 용어 주석

- **FOUC (Flash of Unstyled Content)**: 페이지 로드 시 CSS가 적용되기 전 잠깐 스타일 없는 화면이 보이는 현상. 인라인 스크립트로 테마 클래스를 즉시 적용해 방지.
- **`suppressHydrationWarning`**: Next.js에서 서버와 클라이언트의 HTML이 다를 때 발생하는 경고를 억제. 테마 클래스는 JS로만 추가되므로 서버에선 없고 클라이언트에선 있어 불일치가 발생하는데, 이 속성으로 경고 없이 처리.
- **SVG `<mask>`**: SVG 요소의 특정 영역만 보이게 하거나 숨기는 기능. 마스크의 흰색 = 보임, 검정 = 숨김. 오버레이에서는 검정 부분이 투명(구멍)이 되도록 반전 적용.
- **`@variant dark (&:is(.dark *))`**: Tailwind v4에서 클래스 기반 다크 모드를 활성화하는 설정. `.dark` 클래스를 가진 부모 요소 안에서 `dark:` 접두사 클래스가 동작하도록 함.
- **`as const`로 선언된 배열에 항목 추가**: `as const`는 배열을 읽기 전용 리터럴 타입으로 만들지만, 배열 자체를 교체하는 건 가능. 앞에 항목을 추가하면 됨.
